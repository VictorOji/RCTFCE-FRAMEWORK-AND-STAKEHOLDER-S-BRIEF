# RCTFCE Framework and Stakeholder Brief

This repository contains a business analytics and prompt engineering project demonstrating the application of the RCTFCE (Role, Context, Task, Format, Constraints, Example) framework to a retail performance case study.

## Project Overview

A 30-store retail chain has experienced declining performance over the past 18 months. Using the RCTFCE framework, this project develops a structured AI prompt and generates an executive stakeholder brief to help management:

- Identify stores requiring intervention or closure.
- Evaluate product and category performance.
- Re-engage valuable customers.
- Monitor relevant Key Performance Indicators (KPIs).
- Support data-driven decision-making.
